# Traffic Signal

A web-based project to simulate and visualize traffic signals using simple front-end technologies.

## Overview

This project provides a visual representation of traffic signals, aiming to educate or assist in prototyping traffic light logic. It is implemented entirely using HTML, CSS, and JavaScript for easy access and modification.

## Features

- Interactive traffic signal simulation
- Adjustable signal states and timing
- Responsive layout for different devices

## Technology Stack

- **HTML**: Structure of the web app (45.5%)
- **JavaScript**: Signal logic and interactivity (35.6%)
- **CSS**: Styling and layout (18.9%)

## How to Run

1. Clone this repository:
   ```bash
   git clone https://github.com/gnanu9381/Traffic-Signal.git
   ```
2. Open `index.html` in your web browser.
3. Interact with the simulation directly from your browser. No dependencies required.
